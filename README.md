# 🔍 NetScanner

Консольный сканер локальной сети на Python. Обнаруживает активные хосты, определяет их MAC-адреса, производителей сетевых карт и открытые порты.

---

## 📁 Структура проекта

```
src/
├── main.py        # Точка входа, CLI-интерфейс (argparse)
├── scanner.py     # Ядро сканера — ping, MAC, порты
├── vendor.py      # Определение производителя по MAC-адресу
├── report.py      # Сохранение результатов в JSON
├── decos.py       # Декоратор замера времени выполнения
├── database.py    # (в разработке)
└── __init__.py
```

---

## ⚙️ Как работает

1. **Обнаружение хостов** — отправляет ICMP ping на каждый адрес в подсети параллельно через `ThreadPoolExecutor`
2. **MAC-адрес** — получает через системную утилиту `arp` после успешного пинга
3. **Производитель** — по первым трём октетам MAC ищет совпадение в локальной базе OUI
4. **Сканирование портов** — проверяет список популярных портов через TCP-соединение, тоже в несколько потоков
5. **Отчёт** — сохраняет всё в `reports/scan_report.json`

## 🚀 Запуск

### Сканирование сети

```bash
python main.py scan 192.168.1.0/24
```

С указанием конкретных портов:

```bash
python main.py scan 192.168.1.0/24 --ports 22 80 443
```

### История сканирований

```bash
python main.py history
```

> ⚠️ Функция истории пока не реализована.

## 📦 Зависимости

Проект использует только стандартную библиотеку Python — сторонних пакетов не требуется.

| Модуль | Назначение |
|---|---|
| `subprocess` | Вызов `ping` и `arp` |
| `socket` | TCP-проверка портов |
| `ipaddress` | Парсинг CIDR-нотации |
| `concurrent.futures` | Многопоточность |
| `json`, `datetime`, `os` | Сохранение отчётов |

## ⚡ Производительность

- Пинг хостов — до **50 потоков** одновременно
- Сканирование портов — до **20 потоков** на хост
- Таймаут пинга — **1 сек**, таймаут порта — **0.5 сек**

Параметры можно изменить при создании объекта `Scanner`:

```python
s = Scanner(timeout=2.0, max_workers=100)
```
